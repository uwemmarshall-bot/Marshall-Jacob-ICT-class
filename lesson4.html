What is an HTML Table?

An HTML table is used to arrange information into rows and columns. Tables are useful for displaying timetables, student records, examination results, financial reports and many other forms of organized information.

Main Table Tags

Tag Meaning
<table> Creates a table
<tr> Creates a row
<th> Creates a heading cell
<td> Creates a normal data cell

Example of a Table


<table border="1">

<tr>
<th>Name</th>
<th>Age</th>
</tr>

<tr>
<td>John</td>
<td>16</td>
</tr>

</table>

Adding Images

Images make websites attractive and informative. The HTML image tag is: 


<img src="photo.jpg"
width="300"
height="200"
alt="Photo">

src specifies the image location. width specifies image width. height specifies image height. alt displays text if the image cannot load.

Creating Hyperlinks

Hyperlinks connect one webpage to another webpage or website. Users click hyperlinks to navigate through websites. 


<a href="https://www.google.com">

Visit Google

</a>
