WEB DESIGN & DEVELOPMENT

Lesson 1 - Introduction to Web Design and Development

Lesson Objectives

At the end of this lesson students should be able to:

  • Define Web Design.
  • Define Web Development.
  • Understand websites and webpages.
  • Know types of websites.
  • Know tools used for web development.
  • Create their first webpage.

What is Web Design?

Web Design is the process of planning and creating the visual appearance of a website. It includes colors, fonts, layouts, images and user experience.

Examples

  • School Website
  • Company Website
  • Bank Website
  • Hospital Website
  • E-commerce Website

What is Web Development?

Web Development is the process of building websites using programming languages. Developers use HTML, CSS and JavaScript to create webpages.

Difference Between Web Design and Web Development

Web Design Web Development
Deals with appearance Deals with functionality
Uses layouts and colors Uses programming
UI/UX Logic and coding

Languages Used in Web Development

  • HTML - Structure
  • CSS - Styling
  • JavaScript - Interaction
  • PHP
  • Python
  • NodeJS
