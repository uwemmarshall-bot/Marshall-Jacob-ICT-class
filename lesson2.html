What is HTML?

HTML means HyperText Markup Language. It is the standard language used for creating webpages. Every website on the internet starts with HTML. HTML is not a programming language. It is a markup language that tells browsers how to display content.

HTML uses tags enclosed inside angle brackets (< >) to define elements on a webpage.

Basic HTML Structure


<!DOCTYPE html>

<html>

<head>

<title>My Website</title>

</head>

<body>

Content Goes Here

</body>

</html>

Explanation

  • DOCTYPE tells browser this is HTML5.
  • html is the root element.
  • head contains information about the page.
  • title appears on browser tab.
  • body contains everything users can see.

Common HTML Tags

Tag Purpose
<h1> Main Heading
<p> Paragraph
<img> Insert Image
<a> Create Link
<ul> Unordered List
<ol> Ordered List
<br> Line Break

Example of HTML Code


<h1>My First Website</h1>

<p>Welcome to my webpage.</p>

<ul>

<li>Apple</li>

<li>Orange</li>

</ul>
