What is an HTML Element?

An HTML element is everything from the opening tag to the closing tag. Elements tell the browser what should appear on the webpage. An element normally consists of: 


Opening Tag + Content + Closing Tag

Example:


<p>Welcome to ICT Class</p>

The opening tag is <p> while the closing tag is </p>. The content is "Welcome to ICT Class".

What is an HTML Attribute?

Attributes provide additional information about HTML elements. Attributes are always written inside the opening tag. They usually come as: 


attribute="value"

Example:


<img src="logo.png" width="200">

Here, src tells the browser where the image is located while width tells the browser how wide the image should appear.

Common HTML Attributes

Attribute Purpose
src Specifies image location
href Specifies hyperlink destination
width Image width
height Image height
title Tooltip information
alt Alternative text for images
