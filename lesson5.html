What is an HTML Form?

An HTML Form is used to collect information from users. Forms are commonly used on websites for registration, login, contact pages, surveys, online applications, and online shopping. Information entered into a form can later be processed and stored in a database.

The <form> Tag

Every HTML form begins with the <form> tag and ends with the </form> tag. All input elements must be placed inside the form. 


<form>

Form Elements Here

</form>

Common Form Elements

Element Purpose
input type="text" Text Field
input type="password" Password Box
input type="email" Email Address
input type="radio" Select One Option
input type="checkbox" Select Multiple Options
select Drop-down List
textarea Large Text Area
button Submit Button

Simple Registration Form


<form>

Name:

<input type="text">

Password:

<input type="password">

<button>

Submit

</button>

</form>

Explanation

  • Text Box allows users to type text.
  • Password Box hides typed characters.
  • Radio Button allows one selection.
  • Checkbox allows multiple selections.
  • Select creates a drop-down menu.
  • Textarea allows long messages.
